# ggsegDestrieux: Destrieux Atlas for the 'ggseg' Ecosystem

Destrieux cortical atlas (aparc.a2009s) for the 'ggseg' ecosystem.
Provides a 'ggseg_atlas' object with 2D polygon geometry for use with
'ggseg'.
